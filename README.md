# DebtCommand

DebtCommand is a private monthly debt and bill management dashboard.

## Purpose

DebtCommand tracks income, debts, bills, due dates, minimum payments, credit limits, payment status, and leftover money after monthly minimum payments are covered.

## Stack

- React
- Vite
- TypeScript
- Node.js
- Express
- Prisma
- PostgreSQL
- JWT Authentication
